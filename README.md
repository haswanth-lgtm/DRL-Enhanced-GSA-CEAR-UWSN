# DRL-Enhanced GSA-CEAR for UWSNs

## Project Overview

This project proposes an intelligent routing framework for Underwater Wireless Sensor Networks (UWSNs) using:

- Deep Reinforcement Learning (DRL)
- Gravitational Search Algorithm (GSA)
- Adaptive Clustering
- Data Compression
- Energy-Aware Routing

The objective is to improve network lifetime, reduce energy consumption, and enhance packet delivery performance.

## Technologies Used

- Python
- NumPy
- Matplotlib
- Pandas
- Deep Reinforcement Learning
- Gravitational Search Algorithm

## Features

- Energy Efficient Routing
- Cluster Head Selection
- Residual Energy Analysis
- Dead Node Analysis
- Packet Delivery Ratio Evaluation
- Throughput Analysis
